# 💰 Controle Financeiro de Casal

Um web app progressivo (PWA) construído com Next.js para ajudar na gestão inteligente das finanças a dois.

## 🎯 O Objetivo
O projeto nasceu da necessidade de controlar os gastos do final de semana (como iFood, restaurantes e passeios) e os custos de logística (Uber), garantindo que o casal aproveite o tempo juntos sem estourar o teto de gastos mensal. O app permite lançar despesas rapidamente e acompanhar o saldo restante do módulo de lazer.

## 🛠️ Tecnologias Utilizadas
* **Framework:** Next.js (App Router)
* **Estilização:** Tailwind CSS
* **Distribuição:** PWA (Progressive Web App) para instalação nativa no iOS via Safari.
* **Hospedagem:** Vercel
